Practice Questions

Q1) In the layer hierarchy as the data packet moves from the upper to the lower layers, headers are
a) Added
b) Removed
c) Modified
a) Added
Q2) In asynchronous serial communication the physical layer provides
a) flow control
b) both start & stop signalling and flow control
c) only start signalling
b) both start & stop signalling and flow control
Q3) The physical layer is responsible for
a) line coding
b) channel coding
c) all of the mentioned
c) all of the mentioned
Q4) A single channel is shared by multiple signals by
a) digital modulation
b) multiplexing
c) phase modulation
b) multiplexing
Q5) The network layer is concerned with __________ of data.
a) bits
b) frames
c) packets
c) packets
Q6)Which one of the following is not a function of network layer?
a) error control
b) routing
c) inter-networking
a) error control
Q7)In virtual circuit network each packet contains
a) full source and destination address
b) a short VC number
c) only destination address
b) a short VC number
Q8) A subset of a network that includes all the routers but contains no loops is called as
a) spanning tree
b) spider tree
c) special tree
a) spanning tree
Q9) ICMP is primarily used for
a) error and diagnostic functions
b) addressing
c) routing
a) error and diagnostic functions
Q10) The network layer protocol for internet is
a) ethernet
b) internet protocol
c) file transfer protocol

Q11) Which one of the following algorithm is not used for congestion control?
a) traffic aware routing
b) load shedding
c) routing information protocol

Q12)Which of the following are transport layer protocols used in networking?
a) TCP and FTP
b) FTP and UDP
c) TCP and UDP

Q13) An endpoint of an inter-process communication flow across a computer network is called
a) socket
b) pipe
c) port

Q14) A _____ is a TCP name for a transport service access point.
a) port
b) pipe
c) node

Q15) Two devices are in network if
a) a process in one device is able to exchange information with a process in another device
b) a process is running on both devices
